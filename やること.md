```bash
cd /Users/yudai/Desktop/kintai-app

```

## 2. Vercel でやること

1. https://vercel.com を開く
2. プロジェクト **kintai-app** をクリック
3. **Settings** → 左の **Environment Variables**
4. 次の4つを「Add New」で追加（Key をコピペ、Value は .env からコピー or 下の値、Production にチェック → Save）

| Key（コピペ） | Value（どこから） |
|---------------|------------------|
| `DATABASE_URL` | .env の DATABASE_URL の値 |
| `DIRECT_URL` | .env の DIRECT_URL の値 |
| `AUTH_SECRET` | .env の AUTH_SECRET の値 |
| `AUTH_URL` | `https://kintai-app-khaki.vercel.app` をそのまま |

5. **Deployments** タブ → 一番上のデプロイの **⋯** → **Redeploy** → 完了まで待つ

## 3. 確認

ブラウザで **https://kintai-app-khaki.vercel.app** を開く。ログインできればOK。
